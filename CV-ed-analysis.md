## Weaknesses & Gaps

### 🔴 Critical: TypeScript absent from entire CV
No TypeScript anywhere — not in Skills, not in any job bullet. In 2026 this is **table-stakes for every senior frontend role**. Either:
- He didn't use it (concerning for a senior engineer at Binance)
- He forgot to list it (fixable immediately)
Must clarify and add if applicable. Without it, ATS systems will filter him out of most senior React roles.

### 🔴 Critical: 5-month employment gap (Dec 2025 → present)
No explanation. Recruiters will ask. Needs a one-line narrative ready: sabbatical, startup exploration, personal circumstance — anything is better than silence.

### 🟡 Warning: "Restricted regions / network blocking" pattern
This phrase appears in **three places**: Key Highlights, Binance bullets, Reputation Software bullets. Binance = crypto (understandable context). Reputation Software = unknown. "OTA updates, WebView resource packaging to bypass network blocking, PIP streaming" suggests a platform operating in mainland China or targeting markets with internet restrictions — common in crypto/gambling/adult entertainment. Some enterprise recruiters will flag this.

**Recommendation:** Keep at Binance (brand justifies context). Reframe at Reputation Software toward architecture and delivery, not connectivity bypass.

### 🟡 Warning: "Reputation Software Ltd." is unrecognizable
The English company name doesn't match any known brand. Chinese name (天譽軟體) is also obscure. The product domain (streaming, OTA, multi-region) is not disclosed. Recruiters can't verify the scale claims. Needs a one-line company descriptor: e.g., "B2C streaming platform serving [X] markets, [Y] MAU."

### 🟡 Warning: No TypeScript, no Next.js, no modern testing
In 2026 frontend market, the standard senior stack is React + TypeScript + Next.js + Vitest/Testing Library. The CV shows React + JavaScript + Jest (Puppeteer-only, no unit testing culture mentioned). This positions him behind the market standard.

### 🟠 Minor: "up to 80%" used three times
Profile → Key Highlights → Binance bullet. Reads repetitive. Consolidate: state the metric once in Key Highlights, remove from Profile summary.

### 🟠 Minor: Telexpress (Jan 2018 – Mar 2019) is thin
Only 3 generic bullets. If this can't be strengthened, consider compressing to a single line under an "Early Career" section to reduce noise.

### 🟠 Minor: No LinkedIn, no GitHub, no portfolio link
Senior engineers are expected to have at least one visible public presence. A GitHub with OSS contributions or a personal project would materially strengthen the CV.

### 🟠 Minor: No AI/LLM integration experience mentioned
2026: AI-adjacent experience (copilot tooling, LLM API integration, AI-assisted dev workflow) is increasingly expected. Even noting "adopted GitHub Copilot" signals awareness.
