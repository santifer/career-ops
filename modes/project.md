# Mode: project — Portfolio Project Evaluation

Score across 6 dimensions (1-5):

| Dimension | Weight | 5 = ... | 1 = ... |
|-----------|--------|---------|---------|
| Signal for target roles | 25% | Directly demonstrates the JD skill | Unrelated |
| Uniqueness | 20% | Almost nobody has built this | Everybody has it |
| Demo-ability | 20% | Live demo in 2 min | Code only, no visual proof |
| Metric potential | 15% | Clear metrics (latency, cost, accuracy) | No measurable outcome |
| Time to MVP | 10% | 1 week | 3+ months |
| STAR story potential | 10% | Rich story with trade-offs | Pure implementation only |

## "Interview Pack" requirements

For each approved project:
1. **One-pager**: product + architecture + metrics + evaluation plan
2. **Demo**: live URL or a recorded 2-minute walkthrough
3. **Postmortem**: what worked, what failed, and the mitigations

## 80/20 plan

- Week 1 -> MVP with the core metric
- Week 2 -> polish + interview pack

## Verdicts

- **BUILD** -> plan with weekly milestones
- **SKIP** -> explain why and what to do instead
- **PIVOT TO [alternative]** -> a higher-impact variant
