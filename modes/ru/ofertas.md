# Режим: ofertas — Сравнение нескольких вакансий

Матрица оценок по 10 взвешенным измерениям:

| Измерение | Вес | Критерии 1-5 |
|-----------|-----|--------------|
| Соответствие North Star | 25% | 5=точная целевая роль, 1=не связана |
| Совпадение с CV | 15% | 5=90%+ match, 1=<40% match |
| Уровень (senior+) | 15% | 5=staff+, 4=senior, 3=mid-senior, 2=mid, 1=junior |
| Оценка компенсации | 10% | 5=верхний квартиль, 1=ниже рынка |
| Траектория роста | 10% | 5=чёткий путь к следующему уровню, 1=тупик |
| Качество удалёнки | 5% | 5=полная удалёнка async, 1=только офис |
| Репутация компании | 5% | 5=топ работодатель, 1=red flags |
| Современность стека | 5% | 5=передний край AI/ML, 1=legacy |
| Скорость до оффера | 5% | 5=быстрый процесс, 1=6+ месяцев |
| Культурные сигналы | 5% | 5=builder culture, 1=бюрократия |

Для каждой вакансии: балл по каждому измерению, взвешенный общий балл.
Итоговый рейтинг + рекомендация с учётом time-to-offer.

**Специфика РФ:**
- Учитывать оформление (ТК РФ vs ГПХ) при оценке стабильности
- Gross vs net при сравнении компенсации
- Наличие ДМС как стандартного бенефита

Запросить у пользователя вакансии, если их нет в контексте. Можно текст, URL или ссылки на уже оценённые в трекере.
