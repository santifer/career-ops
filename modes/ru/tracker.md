# Режим: tracker — Трекер откликов

Прочитать и показать `data/applications.md`.

**Формат трекера:**
```markdown
| # | Дата | Компания | Роль | Балл | Статус | PDF | Отчёт |
```

Возможные статусы: `Evaluated` → `Applied` → `Responded` → `Interview` → `Offer` / `Rejected` / `Discarded` / `SKIP`

- `Applied` = кандидат отправил отклик
- `Responded` = компания ответила (inbound)
- `Interview` = активный процесс собеседований

Если пользователь просит обновить статус — отредактировать соответствующую строку.

Показать также статистику:
- Всего откликов
- По статусам
- Средний балл
- % с PDF
- % с отчётом

**Дополнительно для российского рынка:**
- Разбивка по площадкам (hh.ru, LinkedIn, напрямую и т.д.)
- Среднее время ответа по статусу `Responded`
