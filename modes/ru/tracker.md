# Режим: tracker — Трекер откликов

Прочитать и показать `data/applications.md`.

**Формат трекера:**
```markdown
| # | Дата | Компания | Роль | Балл | Статус | PDF | Отчёт |
```

Возможные статусы: `Evaluada` → `Aplicado` → `Respondido` → `Entrevista` → `Oferta` / `Rechazado` / `Descartado` / `SKIP`

- `Aplicado` = кандидат отправил отклик
- `Respondido` = компания ответила (inbound)
- `Entrevista` = активный процесс собеседований

Если пользователь просит обновить статус — отредактировать соответствующую строку, используя канонические значения статусов из списка выше.

Показать также статистику:
- Всего откликов
- По статусам
- Средний балл
- % с PDF
- % с отчётом

**Дополнительно для российского рынка:**
- Разбивка по площадкам (hh.ru, LinkedIn, напрямую и т.д.)
- Среднее время ответа по статусу `Responded`
