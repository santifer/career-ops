# /career-ops-project

Arguments: `$ARGUMENTS` (idea seed, or gap from a recent evaluation)

Load context:
1. `modes/_shared.md`
2. `modes/_profile.md`
3. `cv.md`

Read `modes/project.md` and execute it. The mode will:
- Refine the idea into a tightly-scoped project (1-3 weekends)
- Map it to archetypes it signals (AI Platform, Agentic, etc.)
- Propose stack, milestones, the "one big demo moment"
- Suggest how to publish (repo + README + short post + demo video)

Save to `reports/project-{slug}.md`.
