# Cover-letter-in examples (cover-letter-tex mode)

Fictional `main.tex` + `info.tex` + `body.tex` fixture for
`extract-letter-content.mjs` / `patch-letter-content.mjs` /
`check-letter-length.mjs` tests. Do not use real names, emails, or employers
from your own cover letter here.

```bash
node extract-letter-content.mjs examples/cover-letter-tex
```
