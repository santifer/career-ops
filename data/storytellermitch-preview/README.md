## Preview locally

Open `index.html` directly in any browser:


## Voice constraints applied

- First person throughout
- Banned phrases honored: no "leverage" (verb), no "delve," no "synergy," no "utilize," no "deep dive," no "passionate about," no "thought leader," no "results-driven," no "I hope this email finds you well"
- Em dash convention preserved
- Numbers, not adjectives, on every claim

## Mobile / responsive

Breakpoint at 720px — table-of-cards collapses to single column, nav stacks below brand, hero type drops from 64px → 40px, build-card definition list collapses to single column.

## What is NOT in this preview

- A 401-gated `robots.txt` (lives at site root in production)
