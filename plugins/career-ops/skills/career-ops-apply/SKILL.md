# Career-Ops Apply

1. Read `AGENTS.md`, `modes/_shared.md`, and `modes/apply.md`.
2. Use the user’s `cv.md`, `config/profile.yml`, and `article-digest.md` as source material.
3. Draft answers, fill forms, and prepare uploads, but stop before the final submit.
4. Never submit the application on the user’s behalf.
