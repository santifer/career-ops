# Career-Ops Evaluate

1. Read `AGENTS.md` and `DATA_CONTRACT.md`.
2. For raw JD text or a job URL, read `modes/_shared.md` and `modes/auto-pipeline.md`.
3. For evaluation-only requests, read `modes/_shared.md` and `modes/oferta.md`.
4. For multiple offers, switch to `modes/_shared.md` and `modes/ofertas.md`.
5. Use the existing report, PDF, and tracker flow; do not invent a parallel pipeline.
6. Verify live roles with Playwright when verification is required.
