# Playwright MCP Context

`.playwright-mcp/` contains browser-session artifacts: snapshots, console logs, screenshots, downloaded documents, and other inspection outputs.

Treat this directory as private operational evidence, not canonical product documentation.

Use these files only when reconstructing what happened in a browser session or debugging an application form. Do not derive reusable scoring rules, tracker rules, or user preferences from snapshots unless the same information is confirmed in user-layer source files.

Files here may contain personal data, application form content, cookies-adjacent session evidence, or recruiter portal details. Do not publish publicly.
